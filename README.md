# レトロテトリス - 80年代風パズルゲーム

🎮 **[ゲームをプレイする](src/index.html)** 🎮

このリポジトリは、80年代のレトロゲームスタイルで作られたテトリスゲームです。GitHub Copilot を使用して開発されました。

## 🕹️ ゲームの特徴

- **レトロピクセルアート**: 80年代のアーケードゲームを彷彿とさせるドット絵スタイル
- **ネオンカラー**: サイバーパンク風のカラーパレットで目を引くビジュアル
- **本格的なBGM**: コロベイニキ（テトリス主題歌）のワンコーラス完全版
- **スムーズな操作**: レスポンシブなキーボード操作とゲームプレイ
- **完全なゲーム機能**: スコア、レベル、ライン消去、ハードドロップ対応

## 🎯 操作方法

| キー | 機能 |
|------|------|
| ← → | ピースの左右移動 |
| ↓ | 高速落下 |
| ↑ | ピースの回転 |
| Space | ハードドロップ |
| P | ポーズ/再開 |

## 🎨 技術スタック

- **HTML5**: セマンティックマークアップ
- **TailwindCSS**: レスポンシブデザインとユーティリティクラス
- **ES6 JavaScript**: モダンJavaScript機能を活用
- **Web Audio API**: リアルタイム音楽生成
- **Canvas 2D**: 高性能なピクセルアートレンダリング

## 🚀 インストール・実行方法

1. このリポジトリをクローンまたはダウンロード
2. `src/index.html` をWebブラウザで開く
3. 「ゲーム開始」ボタンをクリックしてプレイ開始！

**注意**: モダンブラウザ（Chrome、Firefox、Safari、Edge）での動作を推奨します。

## 🎵 音楽について

ゲーム中に流れるBGMは、テトリスの象徴的な楽曲「コロベイニキ」をWeb Audio APIを使用してリアルタイム生成しています。8ビット風のスクエア波でレトロゲーム特有のサウンドを再現しました。

## 📱 レスポンシブ対応

デスクトップ、タブレット、スマートフォンに対応したレスポンシブデザインを採用しています。

## 🎨 デザインコンセプト

80年代のアーケードゲームとサイバーパンク文化からインスピレーションを得た、ネオンカラーとピクセルアートを特徴とするレトロフューチャーなデザインです。
